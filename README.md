# Bitcraze Homebrew Tap

Homebrew formulae for Bitcraze tools (macOS and Linux).

## Usage

```bash
brew tap bitcraze/tap
brew install <formula>
```

For example, to install the Crazyflie CLI:

```bash
brew tap bitcraze/tap
brew install cfcli
```

## Updating

```bash
brew upgrade <formula>
```

## Available formulae

- `cfcli` — command-line interface for the Crazyflie ([bitcraze/cfcli](https://github.com/bitcraze/cfcli))

Formulae in this tap are updated automatically by each tool's release pipeline.
